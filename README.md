# Study Sync

[Live Demo](https://junaidsheikh-dev.github.io/Study-sync/)

---

## Project Overview

**Study Sync** is a responsive web application designed to help students efficiently manage and access study resources. The platform offers an intuitive user interface and smooth navigation, making it easier for users to organize their study materials and enhance their learning experience.

---

## Features

- Responsive design for mobile, tablet, and desktop
- Easy navigation to study resources
- Organized layout for quick access
- Lightweight and fast-loading interface

---

## Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- GitHub Pages for hosting
